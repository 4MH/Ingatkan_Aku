langkah-langkah Instalasi:
1. Buka file Inku.exe dari program aplikasi
2. Setelah keluar tampilan Inku.exe klik Next
3. Kemudian akan muncul tampilan License Agreement, pilih "I accept the agreement"
4. Apabila tampil Informastion klik next
5. Klik Next pada tampilan Select Destination Location
6. Kemudian klik next sampai tampilan "Select Additional Tasks"
7. Setelah itu akan muncul tampilan "Ready to Install". Klik Install
8. Kemudian klik next pada tampilan "information"
9. Proses Instalasi telah selesai
